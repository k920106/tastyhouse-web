# TastyHouse Web

Next.js 기반의 웹 애플리케이션입니다.

## 개발 환경 설정

### 필수 확장 프로그램

- Prettier - Code formatter
- ESLint
- TypeScript and JavaScript Language Features

### 코드 품질 관리

```bash
# 코드 린팅 및 자동 수정
npm run lint:fix

# 코드 포맷팅
npm run format

# 타입 체크
npm run type-check

# 전체 코드 품질 검사
npm run code-quality
```

### VS Code 설정

프로젝트에 포함된 `.vscode/settings.json`이 자동으로 적용됩니다:

- 저장 시 자동 포맷팅
- ESLint 자동 수정
- TypeScript 자동 import

## 개발 서버 실행

```bash
npm run dev
```

## 빌드

```bash
npm run build
npm start
```
